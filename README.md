# Zerodha MERN Clone 📈

A full-stack Zerodha-inspired trading platform built using the MERN stack (MongoDB, Express, React, Node.js). This project is for educational purposes to understand how trading dashboards and backend APIs work.

---

## 🚀 Features

- User authentication system
- Portfolio dashboard
- Holdings & positions tracking
- Buy / Sell functionality
- REST API backend
- Responsive UI dashboard

---

## 🛠️ Tech Stack

Frontend:
- React.js
- Context API / Hooks
- CSS

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

Other:
- Axios
- Nodemon
- CORS

## ⚙️ Installation & Setup

### 1. Clone repository
git clone https://github.com/anchallll02/zerodha_mern_project.git
cd zerodha-mern

---

### 2. Backend setup
cd backend
npm install

Create .env file:
PORT=3002
MONGO_URI=your_mongodb_connection_string

Run backend:
npm start

OR
npm run dev

---

### 3. Frontend setup
cd ../frontend
npm install
npm start

Frontend runs on:
http://localhost:3000

---

### 4. Dashboard setup (if separate)
cd ../dashboard
npm install
npm start
